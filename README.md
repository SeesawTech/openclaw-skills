# SeesawTech OpenClaw Skills

Internal repository for shared OpenClaw skills used by the SeesawTech team.

## Overview

This repository contains modular skills that extend the capabilities of our AI agents. Skills are self-contained packages providing specialized workflows, tool integrations, and domain expertise.

## Repository Structure

- `skills/`: The primary location for all skill packages.
- `scripts/`: Utility scripts for creating, validating, and packaging skills.

## Getting Started

To use these skills in your local OpenClaw environment:
1. Clone this repository.
2. Add the `skills/` directory to your OpenClaw skills search path (check your gateway config).
